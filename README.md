# 🔴 Dama Italiana con IA

Un gioco di Dama Italiana sviluppato interamente in HTML5, Tailwind CSS e JavaScript. 

### Caratteristiche:
- **Motore IA**: Algoritmo Minimax con potatura Alpha-Beta.
- **Difficoltà Variabile**: 5 livelli, dal principiante al Gran Maestro.
- **Salvataggio**: Supporto LocalStorage per continuare le partite in sospeso.
- **Responsive**: Giocabile su PC e Smartphone.

[Clicca qui per giocare subito!](https://matteoiaconeta1995.github.io/Dama-Italiana-AI/)
